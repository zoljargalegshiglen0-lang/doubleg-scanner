# Report Design v2.2.2

## Зорилго

Report-ыг өөр scanner-ийн хуулбар мэт харагдуулахгүйгээр DoubleG-ийн өөрийн visual identity-тэй болгов.

## Visual language

- Хар + улаан DoubleG palette
- Цагаан/саарал paper background
- Circular risk gauge ашиглахгүй
- Category bar/group header ашиглахгүй
- Case-file cover, evidence ledger cards, coverage trust map ашиглана
- Hardware дангаараа cheat нотолгоо биш гэдгийг DMA gate дээр тодорхой харуулна
- Footer: `DOUBLEG FORENSIC REPORT | by xanny`

## Dynamic data

Бүх score, finding, coverage, evidence, hash, machine мэдээлэл нь ScanResult-оос автоматаар орно.
