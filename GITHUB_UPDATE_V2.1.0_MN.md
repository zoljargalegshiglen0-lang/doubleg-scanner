# DoubleG Scanner v2.1.0 — PDF Report Polish

## Юу өөрчлөгдсөн бэ?

Энэ update нь PDF report-ийг цэвэрлэж, уншихад илүү амар болгосон.

### Шинэ зүйлс

- **Cheat Detection Summary** хэсэг нэмэгдсэн
  - зөвхөн үндсэн detection-үүдийг улаанаар тодруулна
- Бусад browser / installer / kernel / trace мэдээлэл нь тусдаа **neutral review** хэлбэрээр орно
- Доорх шалгалт бүр тусдаа article section-тэй болсон:
  - Browser History
  - Last Activity
  - Network / Data Usage
  - Local Files & Downloads
  - Kernel & Drivers
  - Deleted Traces
  - Microsoft Defender
- PDF дээр **DoubleG logo** болон brand өнгө орно

## GitHub update

1. `DoubleGScanner_v2.1.0_GitHub_ReplaceFiles.zip`-ийг задлана.
2. GitHub Desktop → repository → **Show in Explorer**.
3. Patch доторх file/folder-уудыг repository root руу copy хийнэ.
4. `Replace the files in the destination` сонгоно.
5. Commit message:

```
Update PDF report layout to v2.1.0
```
