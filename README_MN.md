# DoubleG Scanner v1.6.0 — Compact UI + Named Detection

## Гол шинэчлэлт

- Windows-ийн цагаан title bar-ийг авч, DoubleG dark custom title bar болгосон.
- App-ийн хэмжээг 1040×700 болгож compact, rounded, fixed layout хийсэн.
- Хэрэггүй badge/card-уудыг цөөлж Novacs маягийн цэвэр бүтэцтэй болгосон.
- Энгийн unsigned installer-ийг шууд өндөр эрсдэл гэж улайлгах оноог бууруулсан.
- `knownCheatNames` дүрмийн сан нэмсэн.
- Known cheat name + executable/archive + technical evidence давхцвал `DETECTED` гарна.
- Зөвхөн нэр таарвал `REVIEW` эсвэл warning гарна; худал detection-ээс хамгаална.
- PDF-ийн гол тайлбар, verdict, field-үүдийг Англи + Монгол болгосон.
- PDF дээр илэрсэн cheat нэр, төрөл, илрүүлсэн арга, файлын зам, hash, шалтгаан гарна.
